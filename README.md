# What?
A Nix flake to provide a [helipad](https://github.com/Podcastindex-org/helipad) package and NixOS module. Intended for use with [nix-bitcoin](https://github.com/fort-nix/nix-bitcoin).

## Features
- Updates automatically to latest upstream via `update.sh`
- Hardened NixOS module with soundboard and webhook support
- Sound triggers and other triggers are fully supported
